## Build from source

1. Clone the repo
   ```sh
   https://github.com/dash4k/3-pbo-perpustakaan.git
   ```
2. Go to the build directory
   ```sh
   cd /3-pbo-perpustakaan
   ```
3. Make the binary
   ```sh
   ant compile
   ```
4. Run the binary
   ```sh
   ant run
   ```
5. Clean the binary
   ```sh
   ant clean
   ```
